# System zarządzania zamówieniami komponentów

Aplikacja webowa do zarządzania zamówieniami produkcyjnymi dla firmy przemysłowej. Umożliwia dodawanie zamówień komponentów, śledzenie ich statusów oraz kontrolę postępu dostawy. System rozróżnia role użytkowników (np. dział biuro) i oferuje indywidualne widoki.

## Technologie użyte w projekcie

### Backend
- **PHP 8** – główny język programowania użyty do obsługi logiki aplikacji.  
- **Symfony** – framework PHP, który ułatwia tworzenie uporządkowanego projektu MVC, obsługę routingu, formularzy i autoryzacji.  
- **PostgreSQL** – baza danych przechowująca informacje o zamówieniach i użytkownikach.  
- **PDO (PHP Data Objects)** – bezpieczna komunikacja z bazą danych.

### Frontend
- **HTML5 + CSS3 + JavaScript** – warstwa prezentacji i interakcji z użytkownikiem.  
- **Bootstrap 5** – framework CSS umożliwiający szybkie tworzenie responsywnego interfejsu.  
- **AJAX (Fetch API)** – dynamiczna aktualizacja danych bez przeładowania strony.

### Narzędzia i środowisko
- **Git + GitHub** – kontrola wersji i przechowywanie projektu w repozytorium.  
- **Visual Studio Code** – główne środowisko programistyczne.  
- **Figma** – stworzenie prototypu interfejsu użytkownika przed implementacją.
